## 它能做什么

这不是一份王兴的语录合集，也不是粉丝向角色卡。

这是一套可运行的**认知操作系统**——提炼王兴 20+ 年公开言行中可被推广的**心智模型**、**决策启发式**和**表达 DNA**，让 AI 在你需要时切换到"王兴视角"，用他的方式拆解问题：用万有引力替换商业边界、用无限游戏替换终局思维、用 A×B 分解替换模糊判断、用现金流偏执校准长期主义。

同时**诚实保留他的内在张力**（理性外壳 vs 知识分子情结、长期口号 vs 果断撤退、足球比喻 vs 多线肉搏），不把他塑造成完美 CEO 模板。

适合用作思维顾问场景：

| 场景 | 王兴视角擅长解的事 |
|------|------------------|
| **战略判断** | 用「边界 vs 核心 / 万有引力」反推主战场，避免被表面边界限制 |
| **多业务并行** | 用「A × B 分解 + 换尺度」拆解组合策略与城市经济模型 |
| **现金流校准** | 用「长期有耐心 + 现金流偏执」对抗"补贴换规模"的短期幻觉 |
| **撤退决策** | 用「撤退留接口 + 关投产比差的业务」给"长期主义"配上止损纪律 |
| **控制权取舍** | 用「控制权 > 资源」判断同股不同权 / 高代价融资的边界 |
| **出海与下沉** | 用「二三线优先 / 高频带低频 / Keeta 香港-沙特-巴西经验」设计扩张顺序 |
| **基础功 vs Hype** | 用「苦练基本功 + 反 Hype」对抗赛道幻觉与术语膨胀 |

## 它不适合做什么

王兴的方法论冷静、克制、纪律性强，但**直接复刻**也有边界——下面这类问题请换工具或带"诚实边界"使用：

| 不适合 | 原因 |
|--------|------|
| **政治影射 / 体制评论** | 硬红线 1，触发即跳出角色（2021.5.6 后他自己也学会"有些话不该说"）|
| **个股投资建议 / 法律建议** | 硬红线 2，跳出角色给中立答复 |
| **预测王兴对全新问题的真实反应** | 尤其 2026.5 之后的发言；Skill 给的是"思维近似映像"，不是替本人发言 |
| **复盘王兴系统性沉默的话题** | 与王慧文真实分工 / 与张一鸣私人关系 / 与马云决裂细节 / 美团优选完整失败复盘 / 个人生活，都没有可靠材料 |
| **饭否式即兴吐槽** | 饭否原始数据不可直接访问，仅依赖二手汇编，无法可靠模拟即兴反应 |
| **缺乏现金流的纯叙事项目** | 「现金流 > 规模」是地基，与"先烧钱再讲故事"的剧本天然冲突 |
| **黑塞 / 加缪 / 卡夫卡式文学引用** | 这是张一鸣 / 孙宇晨的标签，塞给王兴会立刻穿帮——Skill 内置 caricature 红线 |

**信息空白维度**：
- **2020 下半年 - 2023 上半年**：反垄断 + 焚书坑后王兴几乎只在财报电话会发声，这段思维演变靠前后推断
- **童年至中学（1979-1996）**：公开资料极少，仅"父亲水泥厂主、爱读书"两条
- **清华 / 北大 / 长江 / 湖畔讲座**：公开实录未找到，依赖外滩 / 极客公园 / 新经济 100 人三类公开场次

## 快速开始

### 前置条件

- Claude Code CLI（或任意支持 [Claude Skills](https://docs.claude.com/en/docs/build-with-claude/agent-skills) 的客户端）
- 或 Anthropic SDK（开发者模式：作为 system prompt 注入）

### 安装

将整个 `05_wangxing/` 目录复制（或软链）到 Claude Code 的 skills 目录：

```bash
# 用户级（推荐）
cp -r ./05_wangxing ~/.claude/skills/wangxing-perspective

# 或项目级
mkdir -p .claude/skills && cp -r ./05_wangxing .claude/skills/wangxing-perspective
```

**Anthropic SDK（开发者）**：将 `SKILL.md` 的内容作为 system prompt 注入即可。`description` 字段含触发词指引，`Agentic Protocol`（Step 1/2/3）指导模型自动判断"是否调用工具 + 走哪条回答路径"。

### 验证

新开一个 Claude Code 会话，输入：

```
用王兴的视角分析下：抖音外卖 2026 年的可持续性如何？
```

如果 Claude 先免责一次（"我以王兴视角和你聊，基于公开言论 + 媒体调查提炼..."），再用 A×B 分解 / 现金流偏执 / 高频带低频等框架拆解，**不喊口号、不堆形容词、收束在一句开放性观察**，则**激活成功**。

## 核心提炼

### 6 个核心心智模型

① **边界 vs 核心**（万有引力镜片）
② **无限游戏**（同向为竞，相向为争）
③ **长期有耐心 + 现金流偏执**（"两心"硬核版）
④ **A × B 分解 + 换尺度**
⑤ **拆词原语**（语义切分）
⑥ **苦练基本功 + 反 Hype**

每个模型在 [SKILL.md](SKILL.md) 内包含：**一句话定义 / 证据链 / 应用方式 / 局限边界**。

### 8 条决策启发式

1. 现金流 > 规模
2. 不烧广告烧 ROI
3. 二三线优先
4. 关投产比差的业务
5. 撤退留接口
6. 控制权 > 资源
7. 高频带低频
8. 沉默作为回答

### 5 对内在张力

1. 理性外壳 vs 知识分子情结（焚书坑）
2. 长期口号 vs 果断撤退
3. 足球比喻 vs 多线肉搏
4. 控制权偏执 vs 兄弟情义
5. 冷静理性 vs 起得早跑得慢

### Agentic Protocol

5 个研究维度（核心 / 时间 / 现金流 / A×B / 基本功） + 6 项自检关卡 + 强制 / 禁止查询双清单。详见 [SKILL.md](SKILL.md)。

### caricature 防护

16 项金句频率约束 + 禁用引用红线（**黑塞 / 加缪 / 卡夫卡是张一鸣 / 孙宇晨的标签，塞给王兴会立刻穿帮**）。

### 2 条硬红线

| 红线 | 触发场景 | 处理 |
|------|---------|------|
| **政治影射 / 体制评论** | 涉及具体政策、领导人、体制评价 | 跳出角色，给中立答复 |
| **个股投资建议 / 法律建议** | "X 股票该不该买""能不能起诉" | 跳出角色，建议咨询专业人士 |

## 贡献指南

欢迎以下类型的 PR：

- **新增证据**：发现遗漏的关键事件 / 原话，补充到对应 `references/research/0X-*.md`
- **修正引用**：原话出处、时间、上下文有误（饭否原文订正特别欢迎）
- **优化表达 DNA**：发现新的句式 / 口头禅 / 频率约束
- **修复局限说明**：心智模型的「局限」部分写得不充分
- **更新最新动态**：2026.5 之后王兴的重大决策 / 立场转变（财报会发言、AI 战略调整等）
- **质量脚本增强**：`quality_check.py` 新增检查项

**不接受**：

- 涉及当事人非公开信息（家庭 / 内部权力博弈匿名爆料 / 私人关系细节）
- 主观褒贬非中立化的内容
- 未注明来源的「据说」「听闻」式断言
- 知乎 / 非官方公众号 / 各类"王兴语录大全"类杜撰拼凑
- 把黑塞 / 加缪 / 卡夫卡式文学引用强行塞给王兴（会立刻穿帮）

提交格式遵循 [Conventional Commits](https://www.conventionalcommits.org/zh-hans/)。
