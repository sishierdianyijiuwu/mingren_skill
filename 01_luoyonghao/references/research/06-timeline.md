## 阶段1：早年（1972-2000）

罗永浩自我表述中的关键词："愤怒"、"格格不入"、"被启蒙"、"东游西荡"。这一阶段没有任何线性叙事，几乎所有谋生方式都试过一遍，但同时通过广泛阅读完成了思想底色的塑造。

### 1972
- **7月9日**，罗永浩出生于吉林省延边朝鲜族自治州和龙县。朝鲜族。父亲罗昌珍，曾任和龙县县委书记、延边州副州长、副书记、政协副主席。母亲、姐姐、哥哥均生活于延边。[维基百科 / 新世界时报]
- 出身在朝鲜族干部家庭，家境相对优渥，父亲对其有极大包容，这是他敢于"中途退出体制"的底层支持。

### 1978-1983：和龙县胜利小学
- 罗永浩最初就读于和龙县胜利小学。性格上的"愤怒"与"较真"在小学时已经显现，他自述常因不接受老师权威而被罚站。

### 1984：举家迁至延吉
- 全家搬至延吉市，罗永浩转进延吉市北山小学。延吉的城市感与少年阅读经验对其影响开始增强。

### 1985-1987：延吉市第六中学（初中）
- 阅读开始进入"野生少年"状态：杂书、武侠、外国文学一并吞下。

### 1988-1989：延边二中（高中）
- 进入高中后与教育体制冲突加剧。
- **1989年下半年（高二上学期）**：罗永浩从延边二中退学。他后来反复表述退学理由："自己和教育体制格格不入"。这是其人生第一个不可逆转折点：自此他不再回到正规教育体系，所有知识与世界观完全自学搭建。[维基百科]

### 1990-1999：东游西荡的十年
这一段是罗永浩本人写得最详尽、社会最熟悉的"奋斗前史"。按其本人回忆与公开报道，主要谋生方式包括：

- **摆地摊**：在延吉与北京摆过书摊、杂货摊。
- **二手书与旧书贩卖**：曾从北京拉书回延吉卖。
- **羊肉串店**：开过约两个月，亏损关张。
- **倒走私车**：经手过倒卖走私车。
- **韩国贩药材**：去韩国倒卖过药材/保健品。
- **期货**：做过一段时间期货。
- **被传销骗**：参与过非法传销课程半年，自己承认这段经历，并将其作为后来求职信里的反面材料。[GamerSky / 36氪]

#### 思想塑造期（关键阅读）
- 公开承认的启蒙偶像与影响来源包括 **李敖**、**王朔**、**王小波**、《罗马帝国衰亡史》，以及对罗大佑歌词的喜爱。
- 罗永浩自述："李敖年轻时形象是个斗士"，把"做斗士"作为自我形象的核心模板。[环球人物/人民网]
- 退学后罗永浩自述："最感兴趣的是《快速致富论》"——这透露了他理想主义与功利主义并存的本色，不是单一标签的"知识分子"。

### 1999-2000：决定考新东方
- 1990年代末，长期靠零工与小生意维生的罗永浩决定彻底转向教书谋生，目标是北京新东方学校。
- **2000年12月**：罗永浩写下那封后来被广泛引用的"万字求职信"《给我一个机会，我将还你一个新东方最好的老师》寄给俞敏洪。信中点对点回应新东方招聘要求，主动坦白辍学、传销、做杂工的全部经历，并以"我不是想做一个普通的老师，而是想做新东方最好的老师，'之一'也行"作为收束。[GamerSky / 网易 / 36氪]
- 经过两次试讲失败、俞敏洪给到第三次机会后，2001年初入职。

## 阶段2：新东方时期（2001-2006）

这一阶段定义了"老罗"作为公共人物的人设：黑色幽默、敢说真话、对体制充满讽刺、对学生真诚关怀。互联网把课堂的"题外话"放大成了全国级的文化现象。

### 2001
- **正式入职北京新东方**，主讲 GRE 句子填空。
- 入职初期数月仍处于试讲—试岗状态，授课压力极大，罗永浩用"边教边学"撑过去。

### 2002
- 课堂内容里夹杂的"针砭时弊+冷幽默"开始在学生之间形成口碑。

### 2003：老罗语录爆发年
- 学生偷录罗永浩讲课中的"题外话"上传到互联网。这些音质很差的录音被冠以《老罗语录》之名风靡全国，**成为2003年一个奇特的文化现象**。这是中国大陆早期"由用户生成内容（UGC）造就的网红"经典案例。[新浪财经 / 投中网]

### 2004
- 主讲范围扩大，开始进入"老师中的明星"状态。
- 收入与学校地位攀升。

### 2005-2006
- 2005、2006年连续登上**百度年度搜索风云榜**人物榜。[新浪财经]
- 2006年6月，老罗在新东方的月薪已经超过5万元（当时北京普遍水平的若干倍）。
- 此时俞敏洪正在筹备新东方赴美上市（2006年9月7日新东方完成纽交所上市），罗永浩与新东方的体制摩擦也到了顶点。
- **2006年6月，罗永浩从新东方辞职。** 后续他与俞敏洪的关系长期成为公开"恩仇"叙事的素材之一。[新浪/投中网]

> 标志性发言（2006年前后）："老老实实做手艺人。" 与"教书不是我最终要做的事，但我想做最好的那一个。"——为他后来的"工匠/手艺人"叙事埋下伏笔。

## 阶段3：公共知识分子与教育创业（2006-2011）

这一阶段罗永浩同时扮演三种角色：博客平台老板、英语培训学校校长、消费者维权"行为艺术家"。三件事都指向同一个核心：在公共空间寻求话语权。

### 2006
- **7月31日**，罗永浩在北京发起创办**牛博网（bullog.cn）**。创办原因明确：在主流博客平台（如新浪博客）他被频繁屏蔽，索性自建一个由他自己当管理员的博客网站。[维基百科 / 人人都是产品经理 / 腾讯云开发者]
- 第一批签约博主包含 **连岳、王小峰、老六、莫之许、王小山、柴静**；随后陆续吸引 **韩寒、慕容雪村、十年砍柴、梁文道、冯唐、熊培云、周云蓬** 入驻。牛博网成为当时中文互联网最重要的意见领袖聚集地之一。

### 2007
- 牛博网影响力达到顶峰，参与多次公共事件讨论。
- 罗永浩个人继续以演讲、视频形式扩大影响力。

### 2008
- **6月**：罗永浩创办"老罗和他的朋友们教育科技有限公司"（北京市海淀区至圣嘉德培训学校）。
- **7月**：**老罗英语培训学校**在北京正式开始营业。开业当年即亏损300多万元，此后长期亏损，仅2011年获得过盈利。[网易教育]
- 牛博网继续运作，但因汶川地震、奥运等事件后的政治敏感期，已开始承压。

### 2009
- **1月9日下午3时许**，牛博网在中国大陆的服务器被关闭。官方理由是"大量登载时政类有害信息"。[维基百科 / 中国数字时代]
- 牛博网的关闭对罗永浩个人是公共表达上的一次重大打击，他后来多次提及"那是最不敢回忆的日子"。[i黑马]
- 牛博国际继续在境外存在，但已不复影响力。

### 2010
- **4月**：自传式作品 **《我的奋斗》** 由云南人民出版社出版。书名直接挪用希特勒同名著作（罗永浩在序里专门做了讽刺性说明），核心是想表达理想主义的奋斗叙事，并附吉大演讲DVD视频。[豆瓣 / 微信读书 / 169 网易订阅]
- 同年继续做演讲，巡回全国高校，"理想主义者"标签在他身上固化下来。

### 2011：西门子冰箱事件——从教书的人转为"消费维权符号"
- **9月27日**，罗永浩在微博吐槽家中西门子冰箱关不上门，西门子家电官方微博称免费维修但"非质量问题"。罗永浩拒绝接受，开始在微博上密集追打西门子，两个多月发布上百条相关微博。[中新网 / 凤凰网]
- **11月20日**，罗永浩联合左小祖咒、冯唐等人，在北京西门子总部门前现场用铁锤砸毁3台冰箱，要求西门子承认质量问题。这一行为艺术式维权事件全网刷屏，德国媒体也予以跟进报道。[中新网 / 央视 / 凤凰网]
- **12月4日**，西门子中国官方发布"西门子家用电器中国总裁盖尔克"公开信，承认冰箱质量确实存在问题。此战让西门子市值一度蒸发数亿。[新浪科技 2025-09-15 回溯报道]
- 罗永浩自此被网友称为"平西王"。

> 关键自述（2011年环球人物专访）："愤怒是一种生活方式。"——这是该阶段罗永浩最准确的人设标签。[环球人物/人民网]

## 阶段4：锤子科技时期（2012-2019）

最受关注、争议最多、最戏剧化的一段。把一个文人/培训校长改造成了"硬件创业者"，故事曲线从"情怀拉满"到"债务6亿"再到"卖身字节"。

### 2012
- **5月15日**，**锤子科技（北京）有限公司**正式注册成立于北京。公司名"Smartisan"取自 Smart + Artisan，意为"智能手机时代的手艺人"。[Smartisan维基 / MBA智库]

### 2013
- **3月27日**，锤子科技在北京国家会议中心举办**第一代 Smartisan OS 发布会**。这是一款基于 Android 深度定制的 ROM。锤子科技此时还没有任何手机硬件，只有一个 ROM 和一个产品理念。[Smartisan维基]
- "情怀"一词在媒体口径里第一次被贴在锤子身上，源头是这场发布会上的"我们做一个好用的东西就好"。[爱范儿 / 极客公园]

### 2014：锤子 T1 与王自如世纪约架
- **5月20日**，锤子科技在北京国家会议中心发布第一款手机 **Smartisan T1**，搭载 Smartisan OS 1.0。发布会被称作"万字情怀演讲"。同步直播观看人数超 1000万，是当年春晚直播的4倍；罗永浩发布会现场发出的微博在 60 秒内被转发 12000 次。[Smartisan维基 / B站 / 界面新闻]
- T1 凭借对称美学的工业设计获得 **2015年 iF 国际设计奖金奖**，是第一款获得 iF 金奖的国产手机。但因工艺要求过高，量产严重延期，定价 3000+ 元远高于同期国产机。
- **8月27日**，罗永浩 vs **ZEALER 王自如**的"世纪约架"，在优酷直播长达 3 小时，全网围观。罗永浩留下名句："**被包养了就别谈什么独立人格**"。该事件让王自如/ZEALER 中心地位严重受损，老罗的"嘴炮"形象也固化下来。[新浪科技 / 界面新闻]

### 2015：坚果手机下沉
- 锤子推出**坚果（Smartisan U1）**系列，价格下探至 899-1299 元，主打千元机市场。
- **1月**，老罗英语培训学校转手给一家主营北美留学课程的教育机构。罗永浩自此完全离开教培。[网易教育]
- 锤子开始进入"漫长的现金流困境期"。

### 2016
- 锤子推出 **M1 / M1L**，回到中端市场，但销量未达预期。
- 公司开始多轮融资以维持运营。

### 2017：坚果 Pro 与"漂亮得不像实力派"
- **5月9日**，锤子科技在深圳举办春季新品发布会，发布 **坚果 Pro**。罗永浩在发布会现场含泪致谢"锤粉"，留下"**漂亮得不像实力派**"这句经久流传的slogan。[爱范儿 / C114 / 动点科技]
- 同步推出 **Smartisan OS 3.6** 的"闪念胶囊"、"One Step 1.5"、"大爆炸 2.0" 三大创新交互。
- **10月**，发布**坚果 Pro 2**，"漂亮得不像实力派"slogan延续，售价 1799 元起。[C114]

### 2018：鸟巢发布会与 TNT 翻车
- **5月15日**，锤子科技在**北京国家体育场（鸟巢）**举办新品发布会，是中国手机厂商首次在鸟巢办发布会。发布两件产品：**坚果 R1 旗舰手机** 与 **坚果 TNT 工作站**（搭载 Smartisan OS 6.6.6，主打"Touch'N Talk 触摸+语音"）。TNT 显示器版定价 9999 元起。[Smartisan官方PR / 少数派 / 维基]
- 发布会上多次现场演示翻车（语音识别识别不出数字、操作卡顿），媒体口碑大幅反转。"裤子都脱了你给我看这个"成为爱范儿评论标题。[爱范儿 / 极客公园]
- **8月20日**，罗永浩战略投资的快如科技推出社交产品**子弹短信**，上线 7 天即拿 1.5 亿元 A 轮，App Store 免费榜与社交榜第一，日下载 44万。[36氪 / 21世纪经济]
- **8月**，发布 **坚果 Pro 2S**（按用户提供资料中的时间点为8月发布，本时间线沿用）。
- 同年下半年，锤子科技多轮被传"资金链断裂"，员工被遣散，供应商讨债不断。

### 2019：彻底卖身字节
- **1月**，**字节跳动以约1.8亿元收购锤子科技的坚果手机品牌与专利**。原锤子手机核心团队并入字节跳动，新设**新石实验室**（隶属字节跳动旗下北京大眼星空科技有限公司）。空气净化器"畅呼吸"、行李箱"地平线"等非手机资产仍归锤子科技，罗永浩在名义上保留锤子科技 CEO 身份。[澎湃 / 中新网 / 维基 / 21世纪]
- **1月15日**，**子弹短信改名"聊天宝"**，加入"看新闻赚钱""读资讯返钱"等下沉玩法，定位彻底改变。[36氪 / 新浪科技]
- **2月-3月**：罗永浩陆续退出快如科技多个间接控股公司股东行列。**3月5日傍晚，聊天宝团队就地解散**，仅留约 20 名运维人员。198天社交梦碎。[新浪科技 / 36氪 / 21世纪]
- **3月28日**，罗永浩不再担任锤子软件（北京）有限公司法人代表。
- **4月25日**，Smartisan OS 官方新浪微博的认证主体从锤子科技变更为字节跳动旗下大眼星空科技。
- **11月1日**，时任字节跳动新石实验室总裁吴德周（原锤子科技 COO）在北京发布**坚果 Pro 3**，罗永浩不再以创办人身份登台。[中新网]
- 同年，罗永浩**个人对锤子科技剩余的供应商及债权人债务承担连带责任**，这是后来"真还传"故事的起点。

## 阶段5：还债与直播（2020-2022）

罗永浩在这一阶段完成了从"硬件理想主义者"到"商业主播"的人设迁移。最关键的事实：他没有跑路，而是公开承担债务、用直播带货还钱。这件事让他在国民层面重新积累了信用。

### 2020
- **4月1日（愚人节）**，罗永浩在抖音开启**直播带货首秀**，"交个朋友"直播间正式开播。3 小时支付交易额超 **1.1亿元**，累计观看 **4800万+** 人次，首件商品是"小米巨能写中性笔"，5 万存货瞬间抢光。这是中国电商直播史上的标志性事件之一。[第一财经 / 36氪 / 证券时报]
- 创办**北京交个朋友数码科技有限公司**，团队主要由原锤子骨干与抖音侧合作团队组成。
- 公开承诺："**用直播还债**"。罗永浩多次提及自己背负 6 亿元个人债务。[澎湃 / 36氪]

### 2021
- "交个朋友"直播间转向常态化运营，工作日固定时段直播。
- **被执行信息陆续清零**：年底前后罗永浩被法院列为被执行人的多笔执行信息被清零。本人回应："仍在按计划还债，还清后会第一时间告知。"[新浪/证券时报]
- 当年还债进度由罗永浩多次公开披露，数字滚动上升至 4 亿+。

### 2022
- **6月**，罗永浩宣布退出抖音直播间。原账户"罗永浩"改名为"交个朋友直播间"，他的微博也由"交个朋友"公司接管。罗永浩从此从直播一线退居二线，把舞台交给"朱萧木"等一代主播。[时代周刊 / 21世纪]
- **7月10日晚间**，罗永浩在交个朋友直播间宣布新公司，名为 **"Thin Red Line"（细红线）**，主攻 AR/MR 平台。他自称这是"最后一次创业"。["细红线"名字源自 1854 年克里米亚战争中英国 93 高地步兵团排成长红线击退俄国哥萨克骑兵的历史典故，此前已被用于坚果 Pro "细红线特别版"。][新浪科技 / 21世纪经济 / 即刻]
- **11月**，**李亮接任"交个朋友"CEO**，全面负责直播运营。罗永浩在管理层上彻底退出。[时代周刊 / 投中网]

## 阶段6：新创业方向（2022-至今）

罗永浩的第三幕：AR 折戟、转 AI、做播客、与新一代科技创业者同场谈话。从"我要做最后一款硬件"变成"AR 十年内都商业化不了"。

### 2022（接 7 月细红线官宣之后）
- 12 月初，**细红线科技完成约 5000 万美元天使轮融资**，投资方包括美团/王兴个人投资、紫米雷军等多方。[阿里云创业 / OFweek VR]

### 2023
- 细红线全年聚焦 AR 眼镜原型研发，扩张硬件团队。
- **9 月**：罗永浩在公开采访与社交平台多次表示"AR 是值得用余生 all-in 的方向"。
- 老罗回归直播间偶尔走访，作为代言人露脸，但日常运营基本不参与。
- "交个朋友"年内进入**港股借壳上市**收尾阶段：2023 年 7 月 11 日壳公司**世纪睿科**完成更名为"交个朋友控股"（HK：01450），上市最后一步走完。[证券时报 / 21世纪 / 财联社]

### 2024
- **上半年**，细红线 AR 眼镜进展不顺，行业内对消费级 AR 商业化时间表的预期普遍下调到 5-10 年。
- **下半年**，罗永浩在内部明确战略调整方向：**暂别 AR，转向 AI 软硬一体**。J1 Assistant（产品代号 Jarvis）启动研发。[新浪科技 / DoNews]
- **8 月 26 日**，罗永浩发布**"真还传 2.0"万字长文**披露：截至当时累计还款 **8.24 亿元**（超过此前披露的"6 亿元"债务），新增**国资 6 亿元债务**待还（来自锤子科技国资借款 3 亿 + 国资股权回购 3 亿的合并）。这条文被同时刊发于澎湃、新浪、证券时报、南都等。[证券时报 / 澎湃 / 南都]
- **12 月 26 日**，新浪创事记发文披露细红线最新进展："暂别 AR，迎来 AI Jarvis"。[新浪科技]
- 罗永浩"交个朋友"五周年节点接近，直播间业务进入"AI 主播 + 真人主播双轨"试水期。

### 2025
- **1 月 5 日**：罗永浩 Jarvis 项目首款产品 **J1 Assistant** 在海外（Android + iOS）上线。语音驱动 AI 助理，集 Chatbot + 笔记 + Todo + AI 搜索 + 邮件助理于一体。**目前限制国内 IP，仅支持英文**。被媒体评价为"大锅乱炖、锤味儿满满，但平淡"。[OSCHINA / 36氪 / 53AI / 腾讯新闻]
- **3 月 8 日**："交个朋友"抖音直播间借 **DeepSeek 大模型 + 数字人罗永浩 + AI 助播朱萧木**单日 GMV **突破 3.3 亿元**，刷新自身记录。背后是 AI 话术自动生成 + 数字人 7×24h 直播体系。[财富 / 36氪]
- **3-4 月**：媒体披露 **细红线硬件团队全部裁撤**，仅留 20 多名软件工程师。罗永浩本人在 21 世纪经济报道访谈中确认转型："**烧了很多钱，证实十年内 AR 眼镜都不太可能商业化。**"招聘聚焦 AI 大模型、多模态交互、AI 操作系统（AIOS）。挖来**小米前 50 号员工、操作系统老兵汪文俊**。[OFweek VR / 36氪智能涌现 / 泰伯网 / 腾讯新闻 / 21世纪经济报道]

## 思想转折点梳理

把罗永浩的人生切成 6 段会过于事件化。若把这 54 年压缩成几次"世界观转弯"，可以更准确地描述他这个人。

| # | 时间 | 转折点 | 关键变化 |
|---|------|--------|---------|
| T1 | 1989 | **高二退学** | 从体制内学生彻底变为体制外个体。决定罗永浩此后所有的"反建制"叙事都不是表演，而是其个人生存模式的真实复刻。 |
| T2 | 1990s | **被李敖、王朔、王小波塑造** | 从"愤怒的失学少年"获得了一套表达"愤怒"的语言系统。"做斗士、做手艺人、不接受权威"这一组人设的语言基础在此奠定。 |
| T3 | 2000 | **万字求职信入新东方** | 从底层小生意人转型为公共人物。第一次把"自我营销/自我袒露"作为生存策略，并成功。这一打法之后会被他在锤子、还债直播、AI 创业、播客中反复使用。 |
| T4 | 2006 | **辞职新东方 + 创办牛博网** | 从教师变成公共知识分子/平台运营者。这一阶段他第一次试图通过组织一个内容平台来放大自己的影响力。 |
| T5 | 2011 | **砸西门子冰箱** | 罗永浩"以个人 IP 对抗大公司"的方法论正式形成。这套方法 2025 年完整复现在西贝事件上——区别只是 2011 年的对手是西门子，2025 年的对手是西贝。 |
| T6 | 2012 | **创办锤子科技** | 从批评者转为造物者。"工匠/Smartisan"叙事让他承担了实际经营责任，也开启了 7 年的现实主义教育。 |
| T7 | 2018-2019 | **鸟巢 TNT 翻车 → 卖身字节 → 子弹短信溃败** | 理想主义叙事第一次被现实主义彻底击穿。罗永浩此后再无"颠覆者"叙事，转向"还债的诚信叙事"。 |
| T8 | 2020 | **抖音直播首秀** | 从"创业者"变为"主播"。一项重大身份妥协，但也是个人 IP 经济变现能力的最大化。 |
| T9 | 2022 | **宣布做 AR（细红线）** | 试图重拾"最后一次创业"的理想主义叙事，但前提条件已经完全变了：他不再赌 all-in，而是同时保留直播、播客、出海培训作为现金流。 |
| T10 | 2024-2025 | **AR 折戟 → 转 AI** | 从"我用余生 all-in AR"自我推翻为"AR 十年内不可能商业化"。罗永浩第一次公开承认自己赌错了方向。**这是他历史上第一次承认"信念错配"，标志着真正进入中年人的现实主义。** |
| T11 | 2025-2026 | **从赌徒转为评论员** | 通过《罗永浩的十字路口》播客与 TikTok Shop 出海培训，他把自己变成了"AI/创业生态的解读者与连接者"，而不是"下场赌一把的人"。 |
