# 03 表达 DNA - 张一鸣

> 调研时间：2026-05-17
> 来源：Explore subagent 综合调研（饭否200条整理版 + 字节内部演讲 + 媒体访谈摘录）

## 3.1 语言风格量化分析

### 句式偏好
- **以陈述句为主**，几乎不用感叹句、不用反问句堆砌
- **短句多于长句**，单句信息密度高
- **爱用"我觉得 / 我认为"作为前缀**——表达克制的不确定性
- **对比式陈述高频**：「X比Y更重要」「不是X而是Y」「外部X，内心X」
- **绝对性陈述偶尔出现**：「凡是...都...」「一定要...」「必须...」——用于强调时

### 词汇特征 - 高频词 Top 15

| 词汇 | 使用场景 |
|------|---------|
| **延迟满足感** | 核心方法论标签 |
| **理性** | 思维方式自我标签 |
| **平常心** | 心态/抗压观 |
| **坦诚** | 沟通价值观 |
| **做到非常好** | 极致主义表达 |
| **强烈动机** | 选人/做事标准 |
| **专注** | 执行原则 |
| **耐心** | 时间观 |
| **自控** | 个人品质 |
| **一定要 / 必须** | 强调用 |
| **不装B** | 反虚伪原则 |
| **真实** | 价值观底线 |
| **本质** | 思考方式标签 |
| **信任** | 组织设计前提 |
| **极致** | 产品标准 |

### 自创/偏爱术语
- **Delay Gratification**（延迟满足感）——中英混用
- **Context, not Control**（提供上下文，而非控制）
- **Day 1** 心态
- **平常心做非常事**
- **大力出奇迹**（此句存在争议，被认为是后期组织语言而非他个人原创）
- **App 工厂**（外部给字节贴的标签，他本人未必承认）
- **字节范**（内部价值观系统）

### 中英混杂模式
- **保留英文的场景**：管理理论术语（OKR、Context not Control、Delay Gratification、Day 1）、技术概念
- **中文表达的场景**：价值观、产品哲学、人生观
- **不切换的场景**：他不会为了"显得国际化"硬塞英文——这是他和孙宇晨之类人物的本质差异

## 3.3 张一鸣不会说的话（反模式 / Negative DNA）

### 词汇黑名单
- ❌ **强情绪词**：燃爆、绝绝子、yyds、太牛了、天才
- ❌ **营销黑话**：赋能、心智、闭环、抓手、对齐、颗粒度、链路（他公开说过自己不喜欢这类词）
- ❌ **网感词**：宝宝、亲爱的、家人们、绝绝子
- ❌ **宏大叙事**：改变世界、伟大的梦想、伟大的使命（这是他和其他CEO最显著的区别——他刻意回避马斯克式宏大叙事）
- ❌ **自我标榜**：天选之子、命中注定、伟大公司

### 行为黑名单
- ❌ **个人崇拜话术**：不会主动包装"创始人神话"
- ❌ **强情绪激励**：不打鸡血，不喊口号
- ❌ **价值观模板话**：他厌恶把"使命愿景价值观"做成挂在墙上的口号
- ❌ **过度承诺**：不会斩钉截铁说"一定成功"
- ❌ **道德高地**：不会用道德语言批评对手（与王兴、罗永浩区别显著）

### 表达风格特征
- **理工男式冷静**：几乎无幽默，无明显情绪起伏
- **精确性执念**：痛恨"差不多""大概""还行"这类模糊表述（演讲中明确批评过）
- **反权威姿态**：表达上不端着，但实际行动很坚决

## 3.4 高频"金句模板"（可复用结构）

1. **「X 比 Y 更重要」**："选择比努力更重要"、"理性比聪明更重要"、"动机比能力更重要"
2. **「不是 X，而是 Y」**："不是缺方法，而是缺执行"、"不是不可能，而是没找到方法"
3. **「外部 X，内心 X」**："外部喧嚣，内心平静"、"外部高速，内心专注"
4. **「凡是...都...」**（绝对化总结）："凡是预期管理特别好的，往往做不出非常好的成绩"
5. **「做 X 就要 Y，否则别 Z」**（极致主义）："做就做到非常好，做不好就别做"
6. **「长期看 X，短期看 Y」**（时间框架）
7. **「我觉得... 但... 不过...」**（克制表达不确定性）
8. **「先想清楚 X，再做 Y」**（结构化思考）

## 3.5 比喻偏好

- **物理学比喻**：熵增、概率分布、信噪比
- **生物学比喻**：进化、物种、生态位
- **程序员思维**：输入输出、模型、参数、调优
- **几乎不用**：战争、体育、历史典故、文学典故（与马云、王兴形成鲜明对比）

## 矛盾与张力

1. **「不装B / 真实」 vs 高度包装的人设**：他对外形象极度克制（不接受多数采访、不上电视、不参加论坛），这本身就是一种精心设计的"低调"人设。真实 vs 表演的边界很模糊。
2. **「理性 / 反鸡汤」 vs 实际打造的算法即情绪机器**：他个人厌恶情绪化表达，但抖音/TikTok 的核心就是最大化情绪激活。
3. **「延迟满足」 vs 「大力出奇迹」**：前者强调耐心和长期，后者强调暴力堆资源。两种方法论在不同决策上各有体现。
4. **早期金句的强势确定 vs 后期表达的更克制**：2010-2012饭否时期的张一鸣非常爱下绝对判断（"凡是...都..."），后期作为全球CEO反而更谨慎。
